# CRUD - REACT - REDUX TOOLKIT - TYPESCRIPT - MUI

### Usage

- Download the api at: https://github.com/rody-huancas/api-crud-node

### Execute the project

- Clone project

  ```bash
    git clone https://github.com/rody-huancas/cliente-crud-ts.git
  ```

- Access the project

  ```bash
    cd cliente-crud-ts
  ```

- Install dependencies

  ```bash
    npm install
  ```

- Running the project
  ```bash
    npm run dev
  ```
